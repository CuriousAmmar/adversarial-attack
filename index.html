What is an adversarial example?

A neural network input crafted with tiny, imperceptible perturbations that cause the model to make wildly wrong predictions — while looking identical to humans.

Attack methods

FGSM — Single step along gradient sign. Fast but weak.

BIM — Iterative FGSM with clipping. Stronger.

PGD — Projected gradient descent. Most powerful iterative method.

C&W — Optimization-based. Finds minimal perturbation.

Defenses

Adversarial training — Include adversarial examples during training.

Input preprocessing — Denoise or transform inputs before inference.

Certified defenses — Provably robust within a perturbation bound.

No defense is universal yet.
